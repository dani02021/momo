# E-commerce
Basic ecommerce website from scratch, with Django.

Tech Stack:
Front: HTML/CSS/JavaScript
Back: Python(Django)

Version: 0.0.1-BETA

# Requirements
1. Python 3
2. Django (>= 3.0.0)
3. PostgreSQL (>= 12)
4. Libraries: six, pycrypto, iso3166, paypalcheckoutsdk, psycopg2

# How to install
1. Install Python 3
2. Install Django
3. Install PostgreSQL
4. Install required libraries
